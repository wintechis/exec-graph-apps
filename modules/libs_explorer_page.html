  • SearchDialog(props: SearchDialogProps): Element
    • Defined in libs/explorer/page/src/lib/search-dialog/search-dialog.tsx:92

    Renders a dialog with search box and list of results. The dialog can provide the user with local results of the current dataset and results from the entire knowledge graph.

    internal

    The results of the local and global searches are stored in the state and initated by effect hooks to make it asynchron and the UI more reactive.
